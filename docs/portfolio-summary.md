# ClarioDocs — Portfolio Summary

## One-Line Description

AI-powered document analysis SaaS — upload documents, get instant AI summaries, risks, insights, and answers.

## GitHub Description

ClarioDocs is a full-stack AI SaaS platform for document analysis. Upload PDFs and text files; the AI extracts content and generates structured summaries, key points, risks, suggested actions, keywords, and document type classification. Users can chat with documents, export analysis as Markdown, and manage subscriptions via Stripe. Built with Next.js 16, TypeScript, Prisma, PostgreSQL, OpenAI, and Stripe.
